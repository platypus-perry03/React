1. Introduce Yourself

React를 활용한 자기소개 카드 프로젝트입니다.  
각 버튼을 클릭하면 해당 인물의 전공 및 관심 기술 정보를 확인할 수 있습니다.

## 🛠 주요 기능
- 자기소개 버튼 그리드
- 클릭 시 모달 창으로 상세 정보 표시
- CSS를 통한 버튼 호버 효과 및 모달 스타일링
